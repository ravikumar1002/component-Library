Badge

Small numerical value, image and status descriptor for UI elements.

Status Badge

In status badge you can use different status like badge-active , badge-busy , badge-offline, you can slao use badge size according to avatar size like badge-empty-round-x-sm, badge-empty-round-sm, badge-empty-round-md, badge-empty-round-lg.

Badge on button and icon

You can use badge size here with these classes: badge-number-round-x-sm, badge-number-round-sm , badge-number-round-md , badge-number-round-lg.

Badge image and number on avatar

For image badge you can use different size like badge-img-x-sm, badge-img-sm, badge-img-md, badge-img-lg with these classes. Number badge, you can use badge-number-round-x-sm, badge-number-round-sm, badge-number-round-md, badge-number-round-lg
