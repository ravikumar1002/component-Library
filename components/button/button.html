Button

Buttons allow users to take actions, and make choices, with a single tap.

Color Button

You can use different color button. To use this button just add btn-primary, btn-secondary, btn-success, btn-info, btn-light .

Types of Button

Here we have different types of button, just add style according to you.

Different Shapes of Button

To use different Shapes of button, add border-squre and border-pill.

Different Sizes of Button

You can also use different size of button according to size need , for this use btn-x-sm, btn-sm, btn-md, btn-lg classes.

Icons Button

Any icon can we use inside the button.

Button with Icon and Text

Position of the icon can be changed. It can be place right or left.

Block Button

For block button just add btn-block class in button.

Float Button

you can style button by just changing btn-float-primary class and also you can select your button corner by adding btn-float-top-right, btn-float-top-left , btn-float-bottom-right , btn-float-bottom-left.
