Avatar

Avatars can be used to represent people or objects. It supports images, Icons, or letters.

Image Avatar

Avatar is available in 3 different sizes. You need to include class avatar-container in div and for size use avatar-sm, avatar-md, avatar-lg.

Avatar Shapes

You can use different shapes of Avatar just add class border-round or border-squre in img tag.

Icon and Text/Number Avatar

You can use different icon, single or double alphabet letters and any number.
